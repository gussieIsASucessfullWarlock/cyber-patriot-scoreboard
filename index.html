CyberPatriot Scoreboard API Documentation

Introduction

Welcome to the CyberPatriot Scoreboard API documentation. This API provides access to data related to the current CyberPatriot competition, including team information, live data, rankings, and graph data.

Endpoints

1. Get Division Data

Endpoint: /api/divisions

  • Method: GET
  • Description: Get information about the divisions, including open, jrotc, and middle divisions.
  • Response:
    • last_updated: Timestamp of when the data was last updated.
    • results: Total number of teams across all divisions.
    • divisions: Division-specific data.

Example

curl https://cyberpatriots.myphone.education/api/divisions

2. Get Ranked Teams Data

Endpoint: /api/ranked

  • Method: GET
  • Description: Get information about ranked teams.
  • Response:
    • last_updated: Timestamp of when the data was last updated.
    • results: Total number of ranked teams.
    • ranked: List of ranked teams with their scores.

Example

curl https://cyberpatriots.myphone.education/api/ranked

3. Get Team Information

Endpoint: /api/team/<team>

  • Method: GET
  • Description: Get information about a specific team.
  • Parameters:
    • <team>: Team ID in the format of xx-xxxx.
  • Response:
    • Team information including team number, score, division, tier, state, play time, score time, image data, and time data.

Example

curl https://cyberpatriots.myphone.education/api/team/12-3456

4. Get Live Team Data

Endpoint: /api/live/team/<team>

  • Method: GET
  • Description: Get live data for a specific team during a CyberPatriot competition.
  • Parameters:
    • <team>: Team ID in the format of xx-xxxx.
  • Response:
    • Live data including team score, division, tier, state, play time, score time, image data, and time data.

Example

curl https://cyberpatriots.myphone.education/api/live/team/12-3456

5. Get Tier Data

Endpoint: /api/tier/<tier>

  • Method: GET
  • Description: Get information about teams in a specific tier (Platinum, Gold, or Silver).
  • Parameters:
    • <tier>: Tier name (Platinum, Gold, or Silver).
  • Response:
    • last_updated: Timestamp of when the data was last updated.
    • results: Total number of teams in the specified tier.
    • tier: List of teams in the specified tier.

Example

curl https://cyberpatriots.myphone.education/api/tier/gold

6. Get Linear Graph Data

Endpoint: /api/graph/linear

  • Method: GET
  • Description: Get linear graph data based on ranked teams' scores.
  • Response: Image file (PNG) containing the linear graph.

Example

curl https://cyberpatriots.myphone.education/api/graph/linear

7. Get Bell Curve Graph Data

Endpoint: /api/graph/bell

  • Method: GET
  • Description: Get bell curve graph data based on ranked teams' scores.
  • Response: Image file (PNG) containing the bell curve graph.

Example

curl https://cyberpatriots.myphone.education/api/graph/bell

8. Get Team Bell Curve Graph Data

Endpoint: /api/graph/bell/<team>

  • Method: GET
  • Description: Get bell curve graph data with a highlighted line for a specific team's score.
  • Parameters:
    • <team>: Team ID in the format of xx-xxxx.
  • Response: Image file (PNG) containing the bell curve graph with a highlighted line for the specified team.

Example

curl https://cyberpatriots.myphone.education/api/graph/bell/12-3456

Error Handling

In case of errors, the API will return a JSON response with an error field providing information about the encountered issue.

Note: Data is updated about every 30 minutes.
